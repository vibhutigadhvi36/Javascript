# ⏳ Countdown & Quotes Application

## 📖 Project Description

This project is developed using HTML, CSS and JavaScript.

It contains three main features:

- ⏳ Countdown Timer
- 💬 Motivational Quotes Slider
- 🪟 Welcome Modal Popup

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📌 JavaScript Concepts Used

- Arrays
- Functions
- DOM Manipulation
- Event Listeners
- Arrow Functions
- Template Literals
- setInterval()
- clearInterval()
- setTimeout()

---

## ✨ Features

✔ Countdown Timer for New Year 2027

✔ Start & Pause Buttons

✔ Automatic Quotes Slider

✔ Previous & Next Buttons

✔ Welcome Popup after 5 seconds

✔ Responsive User Interface
